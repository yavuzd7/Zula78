#### How to add/commit/push

Füge alle Änderungen dem aktuellen Arbeitsschritt hinzu
```bash
git add .
```

Schließe Arbeitsschritt ab 
```bash
git commit -m "Create Readme"
```

Lade alle Arbeitsschritte auf Github
```bash
git push
```

auf Github:
Branch erstellen
